# Smart AI Photo Enhancer

تطبيق ويب احترافي لتحسين الصور باستخدام تقنيات الذكاء الاصطناعي.

## 🎨 التصميم

- **Hybrid Clay-Glass 3D Design**: مزيج من Glassmorphism و Claymorphism
- **Responsive 100%**: يعمل على جميع الأجهزة (Mobile/Tablet/Desktop)
- **RTL عربي كامل**: واجهة عربية احترافية
- **Dark Mode**: واجهة مظلمة عصرية
- **Animated Particles**: خلفية متحركة بجزيئات

## 🚀 البدء السريع

### التثبيت المحلي
```bash
cd smart-photo-enhancer-website
python3 -m http.server 8000
```

ثم افتح المتصفح على: `http://localhost:8000`

### النشر على الويب

#### Netlify
```bash
# تثبيت Netlify CLI
npm install -g netlify-cli

# نشر المشروع
netlify deploy --prod --dir .
```

#### GitHub Pages
```bash
# إنشاء repository جديد
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/smart-ai-photo-enhancer.git
git push -u origin main

# تفعيل GitHub Pages من Settings
```

#### Vercel
```bash
# تثبيت Vercel CLI
npm install -g vercel

# نشر المشروع
vercel --prod
```

#### Cloudflare Pages
1. ارفع المشروع على GitHub
2. اذهب إلى Cloudflare Pages
3. اختر الـ Repository
4. اترك الإعدادات الافتراضية
5. انقر Deploy
