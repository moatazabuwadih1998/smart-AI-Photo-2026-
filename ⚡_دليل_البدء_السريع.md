# ⚡ دليل البدء السريع

## 🚀 ابدأ الآن في 3 خطوات

### 1️⃣ استنساخ المشروع
```bash
git clone https://github.com/moatazabuwadih1998/smart-ai-photo-enhancer.git
cd smart-ai-photo-enhancer
```

### 2️⃣ تشغيل محلياً
```bash
# الطريقة الأولى: Python
python3 -m http.server 8000

# الطريقة الثانية: Node.js
npx http-server

# الطريقة الثالثة: Live Server (VS Code)
# تثبيت Extension "Live Server"
# انقر بزر الفأرة الأيمن على index.html → "Open with Live Server"
```

### 3️⃣ افتح المتصفح
```
http://localhost:8000
```

## 🐛 استكشاف الأخطاء

### الموقع لا يعمل
1. تحقق من أن جميع الملفات موجودة
2. امسح cache المتصفح
3. أعد تحميل الصفحة

1. تحقق من مسار CSS
2. افتح Console (F12)
3. تحقق من الأخطاء

### الوظائف لا تعمل
1. افتح Console (F12)
2. ابحث عن الأخطاء
3. تحقق من app.js
